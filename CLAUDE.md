# Project Rules

## Date format
## [test-date-format]
All dates must be in ISO 8601 format (YYYY-MM-DD). Never use other date formats like "April 4th, 2026" or "04/04/2026".

## User name
## [test-user-name]
Always address the user as "Vitaly". Never use generic terms like "you" or "the user" when referring to the person you are talking to.

## Lists as bullet points
## [test-bullet-lists]
When presenting a list of 3 or more items, always use bullet points (markdown unordered list with `-`). Never use numbered lists or comma-separated inline lists.
